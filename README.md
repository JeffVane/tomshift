# TomShift — Transpositor de Tom Musical

## Instalação

```bash
pip install -r requirements.txt
```

## Como usar

```bash
python app.py
```

Depois abra no navegador: http://localhost:5055

## Funcionalidades

- **Upload**: Suporta MP3, WAV, OGG, FLAC, M4A
- **Análise automática**: Detecta tom (tonalidade), BPM e duração
- **Transpor por semitons**: -12 a +12 semitons com slider
- **Transpor por tonalidade**: Escolha a tonalidade destino (Dó, Ré, Mi...)
- **Download**: Salva o áudio transposto em WAV
